Dupes

During implementation processes we were running into the issue of users having accounts associated to other healthcare systems. This posed an issue as users could possibly miss messages so this allowed me to find the duplicates ahead of time so the Project Teams could make the decisions ahead of Go-Lives.

ADParser

Another application built out of the AD synch process this allows to convert the AD file from JSON format to a readable CSV. This allowed us to easily find issues with user or shared accounts that caused issues with the creation of new users when they may have had a special character that would cause our program to escape from running.
